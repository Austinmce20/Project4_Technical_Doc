What is Python?

Python is an interpreted, high-level, object-oriented programming language. Python shines at being able to create powerful, or simple, tools with fairly easy to read blocks of code known as objects. Python is both free and open-source, which has lead to users from all over the world creating a vast number of libraries and modules. This guide uses Python 3.

Things you may want to know before going further

Overall Python can be a very beginner-friendly language, but some basic computer skills/confidence will help a lot. You should also find a resource to teach you how to either set-up an IDE, or how to run commands in your terminal. To get the most out of this guide, write the code out in your chosen environment and run it.

Your first program

Often when you learn a new language, one of the first things you'll learn how to do is output the statement: "Hello World!" In Python this is really easy to do.

print("Hello World!")

That's right, it's that easy to print in Python! Keep in mind, Python is a case-sensitive language; it can cause issues with your code if you use a capital letter in the wrong spot. Always double check your code!

What does object-oriented mean

Object-oriented languages are great, they allow you to take chunks of code or data, and store it in different forms. One of the simplest objects you'll use in Python is a variable.

x = 600 print(x)

In the above example, you saved 600 in the variable x and called on with the print function.

x = 12 y = 14 z = x + y print(z)

Just because you assign a variable to a value, doesn't mean it's stuck that way forever; you can re-assign variables later on in your code.

x = 60 print(x) x = 20 print(x)
Data Types

Variables can store different data types, and these different data types can be called upon differently and serve different purposes. Python, and programming in general, has many data types. This guide will focus on some of the more simple types.

  • Strings: str: Basic text. "Hello World!"
  • Integers: int: Whole numbers. 6
  • Floats: float: Numbers including decimals. 6.14
  • Boolean: bool: True or False.
If ... else statements

Many languages have the if ... else statements, some in different forms or another. If statements are a way to execute code if certain conditions are present, or to create options depending on the usage. If statements use logical conditions to function 


                x = 3
                if x == 3:
                    print("True")

In the above example, we stored 3 in the variable x, then we used an if statement to print True if x == 3. If we want something to happen when x does not equal 3 (the condition is not true), we would add an else statement.


                x = 4
                if x == 3:
                    print("True")
                else:
                    print("False")

Now that x == 4, the above if else statement would print "False". You can also account for additional conditions with the elif statement.


                x = 4
                if x == 3:
                    print("True")
                elif x == 4:
                    print("Else if")
                else:
                    print("False")

Other conditions you can use include greater than (>), greater than or equal to (>=), less than (<), less than or equal to (<=) and not equals (!=). It's also important to remember that in an if statement, equals to is represented with two equal signs (==).

Loops

Loops are a very powerful and useful tool. The two loops in Python are the for and while loops.

For Loops

For loops iterate over a given sequence, like a range, or list. For loops are good for running a loop a set number of times.


                for i in range(0, 5):
                    print(i)

You might have noticed, it didn't print out a 5; for loops will stop one short of the end of your range. The i in the above code can be changed out for different values, like x or num for example.


                myList = [1, 3, 7, 12, 20]
                for x in myList:
                    print(x)

While Loops

While loops will repeat as long as the specified boolean condition is met.


                x = 0
                while x <= 5:
                    print(x)
                    x += 1

As you saw the above code will keep printing the value of x until x is greater than 5.While loops are great if you need to create an infinite, or never-ending loop. In a command line such as Terminal on Mac or Command Prompt on Windows, you can use ctrl + c to break and force the program to stop; in-case you get stuck in an infinite loop that you can't otherwise stop. 


                x = 0
                while x <= 5:
                    print(x)

Since the above code will never allow x to reach 5, it will loop infinitely.

The break and continue statements

The break statement will break you out of the loop when the specified condition is met.


                x = 0
                while x <= 9:
                    print(x)
                    x += 1
                    if x == 5:
                        break

                print("Final value:", x)

The continue statement is used if you want to skip the code for the current iteration, but want to continue the loop after that iteration. 


                x = 0
                while x <= 9:
                    print(x)
                    x += 1
                    if x == 5:
                        continue

                print("Final value:", x)
Functions

So by now you've already gotten used to calling on some functions, like print() for example... But what if you want to make your own functions? This can be done with the def keyword. 


                def add():
                    num = 2 + 2
                    print(num)

                add()

In the above code we created a function called add: which will add 2 + 2, store it as the variable num, and print it. A function by itself will do nothing until it's called upon, which we do by referencing the function after it is defined. You can store the value returned from a function as a variable.


                def another_function():
                    x = 12
                    while x > 0:
                        print(x)
                        x -= 1
                    return x

                y = another_function()

                print(y)

The return statement in the above code will return the functions result to the element that is calling it. If you do not add a return statement, or leave the value after it blank, it will return a value of None. Any code added within the function after the return statement will be ignored.
